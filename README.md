# What-Can-I-Afford-Today-
A small app that can run locally or from where ever you would like that will data scrape and return averages for goods.

CivicSignal Advanced (Grocery + Fuel + AI)
Features
FastAPI backend
AI-powered scraping
Store detection by ZIP + radius
Fuel + grocery aggregation
Price prediction (basic ML)
Local browser UI
Run Locally
pip install -r requirements.txt
uvicorn main:app --reload
Open:
http://127.0.0.1:8000/static/index.html
